# C Corporation Tax Strategy -- Book Website

Static website for *C Corporation Tax Strategy: How Business Owners Use the 21% Corporate Rate to Build Wealth and Fund Growth* by AE Tax Advisors.

Part of the Entity Tax Strategy series.

## Deployment

Deployed to Vercel at **ccorptaxbook.com**.

## Structure

- `index.html` -- Home/landing page (book details, TOC, audience, series)
- `about.html` -- About AE Tax Advisors
- `404.html` -- Error page
- `css/style.css` -- Styles (navy #1B2A4A / gold #C5A572 design system)
- `js/main.js` -- Mobile nav, smooth scroll, header scroll effect
- `images/` -- Image assets
- `vercel.json` -- Vercel configuration (rewrites, headers, clean URLs)
